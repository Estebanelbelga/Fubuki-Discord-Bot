# Fubuki Discord Bot

## Dependencies

To run this bot, you'll need:
- Python package
    - `discord.py[voice]` 
    - `yt-dlp`
- Installed on your device
    - `ffmpeg`

## Configuring `config.json`

Create a `config.json` file in the root of the project folder with the following content:
```json
{
    "discordToken": "your discord token",
    "prefix": "your bot's prefix"
}
```

## Running the bot

Run the bot with:  
`python main.py`

If the console prints
`'your bot': Bot is up`
then everything should work correctly

> **Note:** This project is **not an official One Punch Man project**
